
A RSS reader.

Requires some hacks to disable browser security so it can do naughty things like read other websites, shock, horror.


	chromium --disable-web-security --user-data-dir=~/.arss --allow-running-insecure-content https://xriss.github.io/arss/


THen it is going to ask you for a special github key so it can keep all its settings in a gist.

Create a key that can read/write gists for your github account from this link.


	https://github.com/settings/tokens/new?scopes=gist


Then you can FEED YOUR ARSS OFF.
